# 4. 章动

本章说明章动的**概念、成因、分类**及与岁差的关系，历史上曾用的模型（如 IAU 1980、Wahr），以及 IAU 2000A 基本角与矩阵约定；最后给出本程序的实现。

---

## 概念

**章动（nutation）** 指地球自转轴在**岁差**基础上叠加的**短周期、小幅摆动**：平赤道（由岁差定义的平滑赤道面）相对惯性空间缓慢转动，而**真赤道**（瞬时赤道面）相对平赤道还有周期从数天到约 18.6 年的晃动，振幅在角秒量级。因此，地轴瞬时方向 = **岁差**（平赤道） + **章动**（真赤道相对平赤道的偏差）。章动使平黄赤交角 ε 变为真黄赤交角 ε + Δε，春分点在黄道上的位置增加 Δψ 等周期项；**真赤道 / 真春分点** = 平赤道 / 平春分点 + 章动改正。

---

## 成因

- **日月对赤道隆起的力矩**：与岁差相同，太阳和月球对地球赤道隆起产生引力矩；但日月相对地球的**位置随时间周期变化**（月球轨道、地月日相对位置），力矩大小与方向随之变化，导致地轴在进动（岁差）之上叠加**周期性摆动**，即章动。
- **月球轨道交点**：月球轨道相对黄道有约 5° 倾角，其升交点沿黄道西退，周期约 **18.6 年**。该周期在日月力矩中占主导，因此章动**主项**周期约为 18.6 年，振幅最大（黄经章动 Δψ 主项约 17″.2，交角章动 Δε 主项约 9″.2）。
- **其他周期**：月球平近点角、平朔望月、太阳平近点角等组合出多种周期（从数天到多年），章动理论用**周期项级数**（幅角为这些平根数的线性组合）表示 Δψ、Δε。

---

## 分类

- **按分量**：  
  - **黄经章动 Δψ**：春分点在黄道上相对平春分点的周期位移（沿黄道方向），主项约 17″.2，周期 18.6 年。  
  - **交角章动（斜章动）Δε**：黄赤交角的周期变化，ε_true = ε_mean + Δε；主项约 9″.2，周期 18.6 年。
- **按周期**：章动项可分为**长周期**（如 18.6 年、半年、月等）与**短周期**（如 14 天、数天）；长、短的分界在不同文献中约 34～90 天。IAU 2000A 等模型给出数百项周期项，每项含 (Δψ, Δε) 的振幅与幅角（基本角的线性组合）。

---

## 与岁差的关系

- **岁差**：地轴方向的**长期、平滑**变化，周期约 25 800 年，用岁差矩阵 P 表示从 J2000.0 平赤道到「当日历元」平赤道的旋转。
- **章动**：叠加在岁差上的**周期摆动**，用章动矩阵 N 表示从「当日历元平赤道」到「当日历元真赤道」的旋转。平→真的完整变换为：**真赤道 = N · 平赤道**（或等价地，先岁差 P 再章动 N）。
- 二者共同决定**瞬时地轴**与**真赤道/真春分点**；视位置、真黄经等均在真赤道/真黄道下计算，故 pipeline 中先施岁差再施章动（见 [9-solar-terms.md](9-solar-terms.md)）。

---

## 章动模型沿革

| 称谓 / 文献 | 说明 | 典型精度 / 项数 |
|-------------|------|------------------|
| **Wahr** | 基于弹性地球与流体地核的章动理论（Wahr, 1981），给出刚体与形变引起的章动幅值，为后续数值序列提供物理基础。 | 理论框架 |
| **IAU 1980** | IAU 采纳的章动序列（1980），基于刚性地球 + Wahr 形变改正，约 106 项（振幅 &gt; 0.0001″）；实用中常取前约 20 项（振幅 &gt; 0.005″）即可达米级定位。 | 约 1–5 mas |
| **IAU 2000A (MHB2000)** | Mathews–Herring–Buffett 等给出的完整章动序列（MHB2000），IAU 2000 采纳；考虑非刚体地球、地核、海洋等，项数约 1300+，精度约 0.2 mas。与 IERS、DE406 等一致。 | 约 0.2 mas |
| **IAU 2000B** | IAU 2000A 的简化版，仅保留 77 项主项，精度约 1 mas，用于对实时性要求高、可接受略降精度的场景。 | 约 1 mas |

- **IAU 1980**：与 IAU 1976 岁差配套，曾长期用于星表与历表；DE200/VSOP87 时代常用。
- **IAU 2000A**：当前高精度标准，与 IAU 2000 章动常数、IAU 2006 岁差（P03）配套；IERS Conventions、SOFA、DE406 等均采用。本程序默认使用完整 IAU 2000A 系数（见下）。
- **IAU 2000B**：77 项截断，本程序在完整数据不可用时回退为此。

---

## 基本角与周期项形式

章动序列中每项的形式为：幅角 θ = n₁·F1 + n₂·F2 + … + n₅·F5（整数系数 nᵢ），对应的 (Δψ, Δε) 为 (Σ Aᵢ sin θᵢ, Σ Bᵢ cos θᵢ) 等。**基本角** F1–F5（Delaunay 幅角）为月球与太阳的平轨道根数组合，与 IERS Conventions 5.43–5.44 一致：

| 符号 | 含义 |
|------|------|
| **F1 = l** | 月球平近点角 |
| **F2 = l′** | 太阳平近点角 |
| **F3 = F** | 月球平纬度辐角（平升交点起算） |
| **F4 = D** | 月球平朔望月（日月平黄经差） |
| **F5 = Ω** | 月球平升交点黄经 |

历元 T = (JD_TT − 2451545)/36525（儒略世纪），F1–F5 为 T 的多项式（平根数），由 IERS 或 SOFA 给出的公式计算。

---

## 矩阵约定与本实现

### 矩阵的书写顺序与含义

与 DE406/IERS、SOFA 一致，章动矩阵写为 **N = R1(ε) R3(−Δψ) R1(−(ε+Δε))**：

- **R1(α)**：绕 **X 轴**（当前架下的 X）右旋 α；**R3(α)**：绕 **Z 轴**右旋 α。
- **书写顺序**（右乘向量时从右到左依次作用）：先 **R1(−(ε+Δε))**，再 **R3(−Δψ)**，再 **R1(ε)**。即对向量 **v** 有 N·v = R1(ε) R3(−Δψ) R1(−(ε+Δε)) v。
- 从**平赤道**到**真赤道**的变换为 **N^T**（即 v_true = N^T v_mean）；因此平→真 = R1(ε+Δε) R3(Δψ) R1(−ε)（与 N 互为转置）。

### 为什么是这个顺序（几何推导）

目标是：从「当日历元**平**赤道、平春分点」到「当日历元**真**赤道、真春分点」。Δψ 是春分点在**黄道面上**的位移（黄经章动），Δε 是**黄赤交角**的周期变化，因此自然分三步在「赤道–黄道」之间切换：

1. **R1(−ε)**（平赤道 → 平黄道）：绕**平春分点方向**（X）转 −ε，使 Z 从平北极变为**黄道极**。此时仍在「平」架下，只是换到以黄道为赤道面。
2. **R3(Δψ)**（在黄道面内转）：绕**黄道极**（Z）转 Δψ，使春分点沿黄道移动，即**黄经章动**。当前 X 已指向**真春分点**方向（在黄道面内）。
3. **R1(ε+Δε)**（黄道 → 真赤道）：绕当前 X（真春分点）转 **(ε+Δε)**，使 Z 从黄道极变为**真北极**，即得到**真赤道**；交角为 ε + Δε，即**交角章动**已包含在内。

因此 **平→真 = R1(ε+Δε) R3(Δψ) R1(−ε)**，与上面 **N^T** 一致。顺序的实质是：**先把平赤道转到黄道（R1(−ε)），在黄道面内做黄经章动（R3(Δψ)），再按真黄赤交角转回真赤道（R1(ε+Δε))**；这样 Δψ、Δε 各自只出现在绕一个轴的旋转里，与 IERS/IAU 对「黄经章动」「交角章动」的定义一致。

### 输出与本实现

- **输出**：平赤道 of date → 真赤道 of date；**ε_true = ε_mean + Δε**。
- **本实现**：默认完整 **IAU 2000A**（`data/IAU2000`）；不可用时回退 **77 项**（IAU 2000B）。历元 T 与岁差一致；代码：MHB2000 / MHB2000Truncated；在 pipeline 中的位置见 [9-solar-terms.md](9-solar-terms.md) 八步之步骤 6。
