# 14. 历表与数据路径总结（VSOP87 / ELPMPP02 / DE406）

所有路径均**相对仓库根**，除非注明为环境变量或绝对路径。下表按「路径 → 各节点：标架、历元、坐标系、版本（星表 / fit DExx）」列出。

**标架简称：** **GCRS** = 地心天球参考系（Geocentric Celestial Reference System），地心为原点、轴与 ICRS 一致；**BCRS** = 质心天球参考系（Barycentric Celestial Reference System），太阳系质心 SSB 为原点、轴与 ICRS 一致。地心 + ICRS 轴即 GCRS；历表 SPK 段在质心/SSB 或 EMB 下即 BCRS 相关。

**时间尺度：每一步按模型要求用 TT 或 TDB，无限制处才可任选。** 管线在每步边界做转换，不沿用上一步尺度。

| 步骤 / 模型 | 规定时间尺度 | 说明 |
|-------------|--------------|------|
| 历表 VSOP87、DE406/DE441 | **TDB** | 质心力学时，历表公式约定 |
| 历表 ELPMPP02 | **TT** | 儒略世纪等公式用 TT |
| DE406 赤道 patch（Vsop87→ICRS 拟合） | **TT** | 经验项 T = (JD−J2000)/365250 为 TT |
| 岁差（P03、Vondrák）、章动（IAU） | **TT** | 标准公式入参 TT |
| 光行时迭代、二分/牛顿的“时刻” | 无硬性规定 | 与历表/岁差步衔接时在边界转换即可 |

TDB↔TT 差约 1.6 ms 量级（TT ≈ TDB − 0.001658 sin(g) 等），定气/定朔秒级精度下在边界显式转换即可。

### 节点：标架、历元、坐标系、版本（定气管线逐步）

时间自变量：光行时 **tr**（由观测 t 迭代得到，tr 对应历表求值时刻）；岁差/章动用 **tr** 对应 JD(**TT**)。历表求值用 **TDB**；岁差、章动公式的入参宜用 **TT**（TT ≈ TDB − 0.001658 sin(g) 等，振幅约 1.6 ms），见下「时间尺度」说明。详见 [9-solar-terms.md](9-solar-terms.md)。

**光行差与管线终点（重要）：** 地心向量采用发光时刻 **tr** 的位置差 **Xproper = x(tr)−x_E(tr)** 时，**周年光行差已折叠进该向量**，故**不得**再施加显式光行差。管线终点定义为：**真黄经（基于 tr 的折叠向量）≡ 视黄经**。若误加显式光行差步，会双重计入，定气可出现约 1.4 s 偏差。

| 步骤 | 时间尺度 | 标架（FK5 / ICRS / GCRS / BCRS） | 历元 | 坐标系（原点 + 架） | 版本 / 星表 / fit |
|------|----------|----------------------------------|------|---------------------|-------------------|
| 1 历表输出 | **TDB** | **FK5 系**：VSOP87 动力学黄道（与 FK5 黄道一致）；**日心**系，非 BCRS | J2000.0 | **日心**为原点，**地球**在日心系下 (L,B,R)，J2000.0 **动力学黄道**；λ_geo = atan2(y,x) | **星表**：VSOP87B（.ear） |
| 2 黄道→赤道 | — | **FK5**：FK5 平赤道（未修正） | J2000.0 | **地心**为原点，**地心太阳**位置，J2000.0 **FK5 平赤道**（黄赤旋转 **R_x(ε₀)**） | FK5 与 VSOP87 动力学黄道约定（Bretagnon & Francou） |
| 3 Frame bias | — | **FK5→GCRS**：**B⁻¹**（标准文献常写 B：ICRS→FK5，故 FK5→GCRS 为 **B⁻¹**） | J2000.0 | **地心**为原点，**地心太阳**，**GCRS**（如 Hilton & Hohenkerk；代码中矩阵即 v_ICRS = R·v_FK5，R = B⁻¹） | 与 ICRS/GCRS 约定一致 |
| 4 DE406 拟合修正 | **TT** | **GCRS**：ICRS 地心架（GCRS / GCRF） | J2000.0 | **地心**为原点，**地心太阳**，**GCRS**（在 B⁻¹ 后施加 **DE406 拟合修正**，Chapront 2000 混合项 Vsop87De406IcrsPatchMixedTerms） | **fit**：**对 DE406 ICRS 拟合**（T = (JD−J2000)/365250 为 TT） |
| 5 转回 FK5 已修正 | — | **GCRS→FK5**：**B**（B⁻¹ 的逆，即标准 B：GCRS→FK5） | J2000.0 | **地心**为原点，**地心太阳**，J2000.0 **FK5 平赤道（已修正）**（便于在 FK5 架上施加岁差） | 与 ICRS↔FK5 约定一致 |
| 6 岁差 | **TT** | **FK5 系**：历元 tr 平赤道（自 FK5 岁差） | 观测历元 **tr** | **地心**为原点，**地心太阳**，**历元 tr 平赤道**（岁差：J2000.0 **FK5** 平赤道 → 该历元平赤道） | **岁差模型**：P03 或 **Vondrák 2011**（可选） |
| 7 章动 | **TT** | **FK5 系**：历元 tr 真赤道（自 FK5 岁差+章动） | 历元 tr | **地心**为原点，**地心太阳**，**历元 tr 真赤道**（章动：平赤道→真赤道，R1(ε)R3(-Δψ)R1(-(ε+Δε))） | **章动**：**IAU 2000A**（SOFA iauNut00a） |
| 8 真黄道 ≡ 视黄道 | — | **FK5 系**：历元 tr 真黄道（即视黄道，无额外光行差步） | 历元 tr | **地心**为原点，**地心太阳**，**历元 tr 真黄道**（真黄赤交角 **R_x(ε)**）；**视黄经** = 真黄经（Xproper 已含光行差） | 定气目标：λ = 0°, 15°, … |

## 14.2 ELPMPP02（月球历表 → 定朔、月视黄经）

### 路径

| 用途 | 路径 / 环境变量 | 说明 |
|------|-----------------|------|
| **默认路径** | `data/elpmpp02` | `DEFAULT_ELPMPP02_PATH`（目录：ELP_MAIN.S1/S2/S3、ELP_PERT.S1/S2/S3） |
| **加载** | `load_all(loader, "data/elpmpp02", Elpmpp02Correction::DE406)` | 测试中写死 `data/elpmpp02` |
| **示例** | `cargo run --example elpmpp02_to_bin -- ../data/elpmpp02` | 从 rust 目录跑时用 `../data/elpmpp02` |

无环境变量；测试/代码中直接写 `data/elpmpp02`。

**时间自变量：光行时 tr（必选）。** 月球距地约 38.4 万 km，光行时约 **1.28 s**，此间月球移动约 **0.7″**；若用观测时刻 t 直接求值会引入几何延迟，定朔可产生约 1.5～2 s 偏差，易突破 GB/T 33661-2017 的 1 s 要求。**必须**先用观测 t 估算距离、得到 **tr = t − d/c**，再将 **tr** 代入历表求值（与 VSOP87 定气一致，迭代得到 tr）。

### 节点：标架、历元、坐标系、版本（定朔/月视黄经管线）

| 步骤 | 时间尺度 | 标架（FK5 / ICRS / GCRS / BCRS） | 历元 | 坐标系（原点 + 架） | 版本 / 星表 / fit |
|------|----------|----------------------------------|------|---------------------|-------------------|
| 1 历表求值 | **TT** | **非 FK5/ICRS**：ELPMPP02_MEAN_LUNAR（历表自有架） | J2000.0 | **地心**为原点，**地心月**平根数位置/速度（**Table 6 相位修正** d_ω 等），ELPMPP02 平均月球架；**入参为光行时 tr**（t → tr 迭代） | **常数**：**DE405** 拟合常数（Table 3 + Table 6）；**Table 6**：Elpmpp02Correction::DE405 或 **DE406**（§4.3.3 与 DE405 同常数）；儒略世纪等公式用 TT |
| 2 Laskar 旋转 | — | **FK5 系**：J2000.0 平黄道（与 FK5 平黄道一致） | J2000.0 | **地心**为原点，**地心月**，J2000.0 **FK5 平黄道**直角 (x,y,z)（**Laskar P,Q** 旋转，平春分平黄道） | 论文 §5，与 J2000 平春分一致 |
| 3 转 ICRS（对比用） | — | **GCRS**：ICRS 地心架（GCRS / GCRF） | J2000.0 | **地心**为原点，**地心月**，**GCRS**（**Table 7**：Rz(φ) Rx(−ε)，**FK5** J2000 平黄道→**ICRS** 轴地心架 = **GCRS**；ε=23°26′21″+0.41100″，φ=−0.05542″） | 对 DE/JPL 的拟合值（elpmpp02 table7），与 DE406 **GCRS** 同架对比 |

## 14.7 实现警示（防回归）

| 风险 | 说明 | 建议 |
|------|------|------|
| **光行差重复** | 管线终点为「真黄经≡视黄经」；Xproper(tr) 已含周年光行差。若在架构/代码中再加一步「显式光行差」，会双重计入，定气偏差约 1.4 s。 | 不增加显式光行差步；与 TDB 对照的「X_r+显式光行差」仅用于验证，不用于生产管线。 |
| **月球光行时缺失** | 定朔若用观测时刻 t 直接求月球位置，未迭代 tr，约 0.7″ 几何延迟 → 定朔约 1.5～2 s 偏差。 | ELPMPP02/DE406 定朔**必须**用光行时 tr 迭代；实现上使用 `use_light_time_moon: true`（或 `ApparentPipelineOptions::pipeline_default()`）。 |
| **Frame bias 方向** | 标准定义 B：ICRS→FK5（X_FK5 = B·X_ICRS）；FK5→GCRS 应为 **B⁻¹**。若代码/文档颠倒，会产生数十毫角秒误差。 | 代码中 FK5→ICRS 的矩阵即 B⁻¹；GCRS→FK5 用 B（或 R^T）。与文献数值一致时注意方向。 |
| **TDB vs TT** | 历表求值用 TDB；岁差/章动公式入参为 TT。TDB−TT 振幅约 1.6 ms，秒级定气定朔可忽略，但高精度宜在岁差/章动前做 TDB→TT。 | 文档与实现中区分：历表入参 TDB；岁差、章动矩阵计算入参 TT。 |
